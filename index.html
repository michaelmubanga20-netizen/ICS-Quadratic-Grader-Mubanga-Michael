# ICS-Quadratic-Grader-Mubanga-Michael A simple web projects that solve quadratic equations and displays results for given coefficients. ## features **Quadratic Equation Solver** -solves equatoins of the form ax^2 + bx + c = 0 -calculates discriminant and determines the nature of the roots -Handles real and complex roots-Imputs validation with friendly error messages **Grading System** -Converts numerical scores (0-100) to letter grades -Uses standard grading scale (A+, A, B+, B, C+, C, D.) -Provides descriptive feedback for each grade range ##How to Run Simply open the 'index.htm' file in any modern browser .No additional setup or dependencies required. ## Teminologies used HTML5 CSS3(with gradient and responsive design) Vanilla JavaScript
